# Automated ETL Data Pipeline with PySpark & Airflow

Production-style ETL pipeline demonstrating automated ingestion, transformation and orchestration of structured datasets using PySpark and Airflow.

Key Features:
- Automated workflow orchestration
- Data transformation using PySpark
- Incremental processing concepts
- Logging and monitoring structure
- Scalable ETL architecture
- Data quality validation concepts

Technology Stack:
- PySpark
- Apache Airflow
- Python
- PostgreSQL
- AWS S3
- Parquet
